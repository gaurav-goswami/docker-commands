â­ Docker Volume is a feature provided by docker that allows us to manage persistent data in Docker containers. When a container is created, it generally runs in an isolated environment with its own file system. However, any data stored within the container is typically non-persistent, meaning it will be lost when the container is stopped or removed. 

â­ Docker volume provides a way to overcome this limitation by allowing us to create a separate storage area that can be shared between the containers or between the containers and the host system.
